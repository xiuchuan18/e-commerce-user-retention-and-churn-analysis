# 电商用户行为分析与用户流失预测

本项目基于 **RetailRocket 电商用户行为数据集**，构建了一个完整的数据科学分析流程，对电商平台用户行为进行分析，并尝试构建用户流失预测模型。

项目主要包含：

* 用户行为分析（EDA）
* 转化漏斗分析（Funnel Analysis）
* 用户留存分析（Retention Analysis）
* Cohort 分析
* 用户流失预测模型（Churn Prediction）
* Streamlit 数据分析 Dashboard

该项目模拟真实互联网公司 **数据分析 / 数据科学项目流程**。

---

# 一、项目背景

在电商平台中，理解用户行为对于优化产品体验和提升转化率至关重要。

常见核心问题包括：

1. 用户从浏览到购买的转化路径如何？
2. 用户留存情况如何？
3. 是否可以通过用户行为预测用户流失？

本项目通过分析用户点击流数据，对用户行为进行系统分析，并尝试构建流失预测模型。

---

# 二、数据集介绍

数据集来源：

RetailRocket Recommender System Dataset

数据为匿名化用户点击流数据，记录用户在电商网站中的行为。

包括三种行为类型：

* view（浏览商品）
* addtocart（加入购物车）
* transaction（购买）

---

## 数据规模

| 指标   | 数值                |
| ---- | ----------------- |
| 总行为数 | 2,755,641         |
| 用户数  | 1,407,580        |
| 商品数  | 235,061           |
| 时间范围 | 2015-05-03 ~ 2015-09-18 |

---

## 行为分布

| 行为类型             | 数量        |
| ---------------- | --------- |
| 浏览(view)         | 2,664,218 |
| 加入购物车(addtocart) | 68,966    |
| 购买(transaction)  | 22,457    |

可以看到：

绝大多数行为为 **商品浏览行为**，只有少部分转化为购买。

这是典型的 **电商转化漏斗结构**。

# 三、用户活跃度分析

统计每日活跃用户（DAU）：

| 指标     | 数值     |
| ------ | ------ |
| 平均 DAU | 11,867 |
| 最大 DAU | 17,516 |
| 最小 DAU | 1,016  |

可以看到用户活跃度存在明显波动，说明平台用户访问具有周期性特征。

# 四、转化漏斗分析

用户购买路径通常为：

浏览 → 加入购物车 → 购买

漏斗统计如下：

| 阶段    | 用户数       | 阶段转化率  | 相对于浏览 |
| ----- | --------- | ------ | ----- |
| 浏览    | 1,404,179 | 100%   | 100%  |
| 加入购物车 | 37,722    | 2.69%  | 2.69% |
| 购买    | 11,719    | 31.07% | 0.83% |

关键发现：

* 浏览 → 加购 转化率 **非常低（2.69%）**
* 加购 → 购买 转化率 **较高（31%）**

说明用户 **主要流失在浏览阶段**。

可能原因：

* 商品推荐不精准
* 商品信息不足
* 用户只是随意浏览

# 五、用户留存分析

用户留存情况如下：

| 时间    | 留存率   |
| ----- | ----- |
| 1天留存  | 2.72% |
| 7天留存  | 0.53% |
| 14天留存 | 0.29% |
| 30天留存 | 0.12% |

可以发现：

用户留存 **下降非常快**。

说明：

绝大部分用户 **只访问一次就不再回来**。

这种情况在电商平台中较为常见。

# 七、用户流失预测

为了预测用户流失情况，构建用户行为特征：

例如：

* 用户生命周期
* 最近活跃时间
* 行为频次
* 活跃天数
* 浏览次数

模型训练窗口：

特征窗口：

2015-08-22 → 2015-09-04

标签窗口：

2015-09-05 → 2015-09-18

如果用户在标签窗口内 **没有行为**，则认为用户流失。

---

## 模型对比

| 模型                  | ROC-AUC | Average Precision | F1    |
| ------------------- | ------- | ----------------- | ----- |
| GBDT                | 0.696   | 0.977             | 0.980 |
| Logistic Regression | 0.695   | 0.977             | 0.979 |
| Random Forest       | 0.685   | 0.976             | 0.980 |
| LightGBM            | 0.662   | 0.973             | 0.980 |

GBDT 模型效果最好。

---
AUC较低的原因：该数据集极端不平衡，如想提高AUC，可通过不均衡采样、SMOTE生成样本等特征工程。

# 八、特征重要性

最重要的用户行为特征：

| 特征       | 重要性   |
| -------- | ----- |
| 用户生命周期   | 0.229 |
| 距离上次活跃天数 | 0.217 |
| 活跃天数     | 0.093 |
| 30天行为次数  | 0.083 |
| 14天行为次数  | 0.081 |

可以发现：

**用户最近是否活跃是最重要的特征。**

# 十二、核心结论

通过本项目可以得到以下关键结论：

1️. 用户主要流失在 **浏览阶段**

2️. 用户留存率较低，大部分用户为 **一次性访问**

3️. 用户最近活跃情况是最重要的流失预测因素

这些发现可以帮助电商平台：

* 优化推荐系统
* 改进商品展示
* 提升用户留存
